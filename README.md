# Shoppable Campaign
[View demo](https://chec-shoppable-campaign-template.netlify.app/#/)

## Project setup

```
yarn
```

### Copy example env and add chec credentials
```
cp example.env .env
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
